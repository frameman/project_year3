## git CLI 指令

### 從github上 下載檔案
```
git clone 'website'
```

### 從自己環境上傳檔案

#### step1
add 檔案到暫存區
```
git add . file_name
```

#### step2
確定自己的檔案有上存到暫存區
```
git status
```

#### step3
上傳到Gitgub
```
git commit -m "註解"
```

#### step4
把檔案上存到github
```
git push origin
```
>可能會遇到問題
> * 需set email 和 username
> * commit 的時候要連動github帳號


### 從githu取得最新的檔案
```
git pull 
```
